  • 個人簡介

    • 彭彭老師和Wehlp Bootcamp 的夥伴們好！ 我是Dennis! 九月將從哥德堡大學傳播碩士畢業！ 個人主要興趣是在Digital Communicaion。碩論研究的主題是關於視覺多媒體的在網站上的共感分析，也因為這個論文把我，身為一個社科院出身的學生，和想成為前端軟體工程師，逐漸拉攏靠近的關鍵！自己也在Coding 和逐漸拼出網頁並在按下執行後的成功找到樂趣，和滿滿成就感！ 未來展望則希望能夠透過前端coding 能力結合數據導向分析能力，在瑞典就業市場中，轉職找到理想的職缺。

  • 為了成為軟體工程師，曾做過什麼努力？若有具體專案作品請分享給我們。

    • 關於前端這個部分，一方面過去真的沒有背景，目前也還沒有專案成果。大部分的練習都是一步一步在Youtube上面看人家怎麼做跟自己Google用關鍵字找資料。畢竟現在真的已經是Youtube大學了！什麼東西都有教，什麼東西都找的到！之前看過的Youtuber 有 Programming with Mosh 的MySQL in 3Hours, 還有Python Tutorial-Python Full Course for Beginners

    另一方面，因為碩論碰到一些前端後，身邊一些工程師朋友們也推薦我使用Codecademy 的Career path 目前的進度大概20%左右！就是一些基礎的HTML 還有CSS 但是我覺得除了一些Challange project 外，很多時候Codecademy都已經把大部分的程式碼寫好了，我們只要按照指示更改一個部分，就會覺得很心虛，就如同彭彭老師說的一樣，好像都是在這邊抄一點那邊抄一點。 最後都沒搞懂邏輯，做起來就很心虛。 因此很希望這次透過很大量的自己動手操作和找文獻資料，把code 無中生有的慢慢寫出來！

  • 如果參與這個計畫，會怎麼安排學習時間？

    • 大方向的時間安排是九月中會進行碩論口試，之後碩班就會告一個段落！並開始倒數一年的找工作簽。換句話說，口試過後若有幸錄取成為WeHelp的學員， 9/19 起我就是直接從全職碩生，立即轉變成為全職轉職的Wehelp學生！而身為全職轉職的學生，就是可以優先的的把60小時的工作量優先排進行程表！ 而這個倒數一年的找工作簽一方面剛好配合上第三屆的WeHelp的期程；另一方面也給我破釜沈舟的壓力，推自己一把。希望在26週後，可以拿出即戰力和作品集，如彭彭老師說的在求職上足以和本科學生抗衡！

    至於60小時的工作量的細節時間分配，目前已知除了會有每週固定一天的實習可能無法有重度Workload以外！剩下時間會比照過去我碩班分配閱讀文獻的和上課的時間分配。過去是早上九點開始上課到下午五點！ 晚餐、運動過後，普遍都是從晚上十點讀文獻到凌晨兩點或到工作達標為止！ 而這套模式在這兩年下來都是可行的，畢竟我相信 壓力是成功的好夥伴！在瑞典的就業市場中找到工作一方面也給了有很大的內驅力！因此我覺得如果可以變成我想望的樣子，這些犧牲是值得的！ 以上就是我目前對於如何參與以及配合課程以及工作時間的安排！

  • 是否有想要加入的軟體公司？為什麼想加入該公司？

    • Spotify

      • 最想加入的一間公司是Spotify! 畢竟他是一個我每天都一定會使用的app。 同時近幾年在年底時都會有一個個人專屬的統計資料，展示每個人聽的音樂類型、時間等等.......而這樣做同時滿足大家想要有有自己個性的音樂品味，在和別人不一樣的同時;也同時找到和自己有一樣音樂喜好的人！ 覺得是一個很有個性的操作！

    • CloudBeds

      • 第二個想加入的應該是Cloudbeds, 過去曾在旅宿業工作過大約三年的時間！覺得Cloudbeds這個系統真的是救星！間單直覺的操作就可以通時串接到各大訂房網站，並跨平台控房和調整價錢，避免超賣！完全是旅宿業神器！！覺得如果能夠開發更多功能例如Data analytic或是爬蟲追蹤同業價格做競品分析，感覺會很有趣！

  • 請描述一件你曾經碰到最困難的事情，如何克服?

    • 最困難的一件事情，我想是應該是我剛來瑞典唸書時，在英文學術寫作上屢戰屢敗。在寫作跟語言邏輯上和中文寫作有蠻大差異，加上過去也不是長時間使用英文。因此很多時候，老師給的評論都是我看得懂你想說什麼，但是你報告這樣寫並沒有很好的表達你的意思。 就退回來重頭寫過！ 當時就覺得這個是必須要立即解決的問題！但是語言這個東西也不是說一天兩天就可以馬上進步的。

    那時候，認清到問題跟障礙所在，接下來就是解決問題 排解問題！ 我也就翻遍所有手邊能找到的線上線下資源！包括Youtube, 網站、學校的學術英文組織、工作坊。 而這其中，我印象最深的線上資源是一個Youtube 頻道 叫作余綺芳的英文頻道！因為線上資源就是可以隨取隨用！當時就是每天晚餐配英文教學影片！並同時把自己抽離中文的邏輯，自己創造一個沈浸式的英文學習環境！ 以及大量聽英語母語人士說話！透過模仿他們說話，把這些東西轉化並內化成自己的東西。

    在大概半年後！在英文寫作上逐漸有起色，不再是永遠忐忑不安的交出作業而是對自自己寫的東西很有信心！但學習大部分都是學無止境，學的越多覺得懂的越少！永遠可以有精進跟挑惕的地方。但經歷這次經驗後，一方面體會到！只要有心幾乎沒有不可能解決不了的問題！ 畢竟現在網路上，真的有太多開放的資源了也深刻體驗到線上資源的便利及有效性！全看自己的決心！
